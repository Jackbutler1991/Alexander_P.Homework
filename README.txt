1) 1.1 Для копирования репозитория на ПК запустить в командной строке команду (предварительно должен быть установлен Git):
Git clone https://github.com/Jackbutler1991/Alexander_P.Homework.git
2) 2.1 Создать пустой репозиторий на локальном компьютере с помощью команды:
Git init
 2.2 Отредактировать проект, создать любые файла/папки в папке нужные для проекта (например README.TXT)
 2.3 В папке проекта можно создать файл .gitignore и добавить его файлы, которые должен не учитывать Git
 2.4 Что бы добавить созданный файл из рабочего каталога в раздел проиндексированных файлов в гите, команда:
Git add . (или Git add readme.txt, что бы добавить конкретный файл)
 2.5 Что бы создать коммит с названием написать команду:
Git commit -m "Название коммита"
 2.6 Что бы создать отдельную ветку в Git и перейти в нее прописать команду:
 git checkout -b  "название ветки"
 2.7 Создать новый репозиторий в GitHub
 2.8 Что бы подключить git к репозиторию GitHub написать команды:
 git config --global user.name "никнейм" и git config --global user.email "имейл"
 2.9 Что бы привязать удаленный репозиторий к локальному:
 git remote add origin "вставить ссылку с гитхаб"
 2.10 Что бы залить изминения на удаленный сервер, команда:
 git push -u origin "название ветки"
